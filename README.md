## ğŸ® How to Play

1. Choose the number of questions.
2. Select difficulty level:

   * `1` Easy
   * `2` Medium
   * `3` Hard
   * `4` Mix
3. Select operation type:

   * `1` Add
   * `2` Subtract
   * `3` Multiply
   * `4` Divide (rounded to 1 decimal)
   * `5` Mix
4. Answer each question.
5. See your result summary at the end.
6. Choose to play again or exit.
